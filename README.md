# AI-SOC – Artificial Intelligence Based Security Operations Center

This project focuses on designing and implementing an AI-powered Security Operations Center (AI-SOC) that uses machine learning techniques to detect and respond to cybersecurity threats in real timemm

## Objectives
- Collect and analyze security logs from multiple sources
- Detect known and unknown cyber attacks using ML
- Reduce false positives and response time
- Provide real-time monitoring and alerting

## Technologies
- Python
- Scikit-learn
- ELK Stack
- Zeek / Filebeat
- Ubuntu Linux

## Project Status
Initial setup and dataset analysis in progress.
